Puslapio struktūra

  • header - Puslapio viršus, kuriame nurodoma info apie puslapį(arba navigacija, sekcija)
  • nav - skirtas kurti puslapio menu
  • section - didžiulė puslapio dalis(didesnė už div)
  • div - maža puslapio dalis
  • main - pagrindinė puslapio dali
  • article - main viduje esanti sekcija
  • aside - šoninė puslapio dalis, sekcija
  • footer - puslapio apačia(rėmėjai, kontaktai, t&c, social linkai, copyrights, kalbos pakeitimas)

Menu info

Kuriant menu naudojamas nav tag'as, kurio viduje yra rikiuotas(arba ne) sąrašas. To sąrašo elementų(li) viduje yra nuorodos(a href) kurios nurodo į kažkurią puslapio dalį arba kitą puslapį.

Nuorodos tag'o(a) atidaromojoje dalyje esantis atributas href nurodo į kur vartotojas turi būti nukeltas paspaudus ant nuorodos.

Jeigu atributas nurodo į kažkokį ID(href=#ID_Pavadinimas), tuomet vartotojas yra nukeliamas 5 puslapio dal5, kuriai yra suteiktas ID(id=ID_Pavadinimas).

Jeigu a href atributas nurodo 5 kitą html failą(href="./aplankas/failas.html"), tai vartotojas nukeliamas į nurodytą puslapį. Pvz.: Random puslapis

Display: Inline, Block ir Inline-Block

Inline

Inline tai tokie elementai, kurie užima tik tiek vietos ekrane, kiek jiems yra reikalinga(koks jų dydis). Minusas, kad negalima keisti dydžio. Pliusas tas, kad jie lengvai įtalpinami tarp kitų informacijos vienetų.
Tag'ų pavyzdžiai: a, span , img, input, output, label, select, textarea, cite ir t.t.

Block

Block elementai užima visą leistiną vietą į šonus ir neįsileidžia kitų elementų į tą pačią eilutę. Minusas, kad užima visą leistiną plotį. Pliusas, kad galima keisti jų dydį
Tag'ų pavyzdžiai: h1-h6, div, section, main, article, nav, ol, ul, li, aside, table, blockquote, canvas ir t.t.

Inline-Block

Inline-Block - tai elementai, kurie pasisavino geriausias linijinių ir blokinių elmentų savybes. Jie užima tik tiek vietos kiek jiems reikia, bet jų dydį galima keisti.

Model box

Margin - atstumas nuo elemento rėmelio kraštų iki aplinkinių elementų rėmelio(skirtingų elementų margin'ai persidengia į viršų ir apačią. Į šonus nepersidengia).

Border - rėmelis, kuris yra tarp margin ir padding.

Paddding - atstumas nuo elemento content(turinio) iki elemento rėmelio kraštų.

Content - elemento turinys. Blokinio elemento turinys užims visą leidžiamą eilutę ar nurodytą dydį.
