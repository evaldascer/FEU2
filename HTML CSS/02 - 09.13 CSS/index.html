CSS rašymo būdai

  • Inline - Linijinis rašomas tag'o viduje naudojant 'style' atributą
  • Internal - Vidinis rašomas style tag'o viduje(head dalyje)
  • External - Išorinis rašomas atskirame css faile, kurį reikia prijungti prie html naudojant link tag'ą

CSS selectoriai

  • Element - parenkamas elementas pagal jo pavadinimą
  • id - sukuriamas id pavadinimas ir priskiriamas vienam elementui(#idPavadinimas{})
  • class - sukuriamas klas4s pavadinimasir priskiriams bet kokiam kiekiui elementų(.klasesVardas{})

Stiliai

  • color - skirtas pakeisti teksto spalvą(color: red)
  • font-size - skirtas pakeisti teksto dydį(font-size: 20px)
  • background-color - skirtas pakeisti fono spalvą(background-color: aqua)
  • font-family - skirtas pakeist šriftą(font-family: monospace)
  • font-weight - skirtas pakeisti teksto storumą(font-weight: bold)
  • line-height - skirtas pakeisti teksto eilutės aukštį(line-height: 20px)
  • border - skirtas nurodyti rėmelio storį, tipą, spalvą(border: 5px solid black)
  • border-radius - skirtas nustatyti rėmelio kampų apvalumą(border-radius: 100px)
  • text-align - skirtas lygiuoti tekstą(text-align: center)
  • font-style - skirtas suteiksti tekstui stilių(italic, normal, oblique)

Spalvos

  • 140 spalvų pavadinimų(red,...)
  • HSL kodas - mažai naudojamas HSL(0, 100%. 50%)
  • HEX kodas - naudojamas (#FF0000)

Permatomos spalvos

  • HSL permatomumas nurodomas nuo 0(permatomas) iki 1(pilnai matomas) (hsla(0. 100%. 50%, 1))
  • RGB permatomumas nurodomas nuo 0(permatomas) iki 1(pilnai matomas) (rgba(255, 0, 0, 1))
  • HEX permatomumas nurodomas nuo 00(permatomas) iki FF(pilnai matomas) (#FF0000FF)

HEX ir RGB

RGB - red green blue. Kiekvienos spalvos nurodomas kiekis(0..255(stiprumas))

HEX - kiekvienos spalvos suteikiama nuo 00(nesuteikiama) iki FF(suteikiama daug)
