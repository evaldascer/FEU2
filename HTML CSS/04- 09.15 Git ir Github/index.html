Position

  • Static - statinė
  • Relative - relatyvioji, giminingoji
  • Absolute - absoliučioji
  • Fixed - fiksuotoji
  • Sticky - lipnioji

Naudojant bet kurią apart statinės pozicijos, gauname prieigą prie 4 naujų stilių: top, left right bottom z-index.
Šie stiliai nurodo per kokį atstumą elementas turi būti atitrauktas.
z-index nurodo elemento prioritetą sluoksniuose esant persidengimui(z ašyje)

Statinė pozicija - nieko ypatingo nedaro. Net nesuteiki galimybės naudoti top/left/right/bottom/z-index.

Absoliučioji pozicija - elementas išeina iš puslapio tėkmės.
Elementas atsitraukia nuo artimiausio tėvinio/senelionio/.. elemento, kuris turi kokią nors poziciją(positio:relative).
Jei neranda, atsitraukia nuo html lango.

Ir fiksuojasi nurodytoje arba pradinėje pozicijoje.
Jam suteikus top/left/right/bottom, per tiek atsitraukia nuo ekrano šonų.

Sticky - lipnioji pozicija iš puslapio tėkmės neišeina. Jis prlimpa prie ekrano kraštų ir neišeina ir tėvinio elemento kraštų. Ekrano kraštams priartėjus prie jo, jis prilimpa prie ekrano krašto per nurodytą top/left/right/bottom(position) atstumą ir juda kartu su ekranu, kol nepasiekia tėvinio elemento ribų.
