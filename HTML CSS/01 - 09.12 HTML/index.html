Tag'ų sąrašas

  • H1-H6 - Antraštės sąrašas
  • p - paragrafas
  • img - paveikslas
  • a - nuoroda
  • div - Divas(konteineris)
  • ul - nerikiuotas sąrašas
  • ol - rikiuotas sąrašas
  • li - sąrašo elementas
  • dl - detalusis(aprašomasis) sąrašas
  • dt - detaliojo(aprašomo) sąrašo elementas
  • dl - detaliojo(aprašomo) sąrašo elemento aprašas
  • hr - horizontali linija
  • br - eilutės lūžis
  • table - Lentelė
  • caption - lentelės pavadinimas
  • colgroup - lentelės stulpelių grupė
  • thead - lentelės galva
  • tbody - lentelės kūnas
  • tfoot - lentelės apačia
  • tr - lentelės eilutė
  • th - table head, lentelės info pavadinimas
  • td - table data, lentelės duomenys
  • form - forma
  • label - etiketė
  • input - įvesties laukas
  • textarea - texto įvedimas
  • button - mygtukas
  • center - centruoja tekstą
  • b, strong - bold tekstas
  • i - paverčia tekstą, italic
  • em - paverčia tekstą, italic
  • sup pakelia ir pamažina tekstą
  • sub - pamažina ir nuleidžia tekstą
  • ins - pabraukia tekstą
  • mark - pažymi tekstą geltonai
  • s - perbraukia tekstą
  • small - sumažina
  • big - padidina
  • span - tuščias linijinis tag'as
  • link
  • style

Atributų sąrašas

  • src - Šaltinis(nuoroda į failus)
  • href - nuoroda į internetą
  • width - plotis
  • height - aukštis
  • onSubmit - formos apteikis(ją pateikus)
  • for - kam skirta etiketė
  • id - elemento ID
  • class - elemento klasė
  • name - formos elemento vardas
  • type - įvesties lauko tipas
  • placeholder - įvesties lauko pavyzdinis tekstas
  • onclick - mygtuko pateikimas(ką daryti pateikus)
  • rel - relation su css
  • style
  • list-style-type - skirtas pakeisti sąrašo žymėjimą

Klaviatūros trumpiniai

  • CTRL Z,X,C,V - atgal, kirpti, kopijuoti, klijuoti
  • CTRL N - naujas langas
  • CTRL O - atidaryti
  • CTRL T - naujas tabas
  • CTRL R(F5) - refresh
  • CTRL A - žymėti viską
  • CTRL S - saugoti
  • SHIFT ALT F - viso ko tabuliavimas
  • SHIFT ALT - kalbos keitimas
  • ALT TAB - lango keitimas
  • ALT SHIFT TAB - lango keitimo navigavimas
  • WIN W - clipboard
  • CTRL + <- / -> - vaikščiojimas per žodžius
